## Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/your-repo.git
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Development Server
```bash
npm run dev
```

Visit localhost:**** in your browser to view the responsive page.
